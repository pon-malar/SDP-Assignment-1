# Task Manager & To-Do List Web Application

##  Project Overview
The Task Manager is a simple and user-friendly web application developed using basic web technologies.  
It helps users organize their daily activities, track completed tasks, write notes, and stay productive.

The application uses a clean and subtle user interface to ensure a calm and focused experience.

---

##  Features
- Welcome home page with centered layout and illustration  
- Add tasks to a to-do list  
- Mark tasks as completed  
- View completed tasks separately  
- Daily focus tips for productivity  
- Task statistics (total & completed tasks)  
- Notes and reminders section  

---

##  Pages Included
1. Home  
2. Add Tasks  
3. Completed Tasks  
4. Daily Focus  
5. Statistics  
6. Notes & Reminders  

---

##  Technologies Used
- **HTML** – For page structure and content  
- **CSS** – For styling and layout design  
- **JavaScript** – For interactivity and DOM manipulation  


---

##  How to Run the Project
1. Open the project folder in **Visual Studio Code**
2. Make sure all files are in the same folder
3. Right-click on `index.html`
4. Select **Open with Live Server**
5. The project will open in your browser
